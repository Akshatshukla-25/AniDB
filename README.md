# AniRate

AniRate is an anime discovery and rating web app. Browse trending and top-rated anime, search by title, view detailed info, and manage your watchlist — all powered by the [Jikan API](https://jikan.moe).

## Pages

- **home.html** — Landing page with hero, trending, and top-rated sections
- **search.html** — Search anime by title
- **detail.html** — Full details for a selected anime
- **watchlist.html** — Your saved watchlist

## Tech Stack

- Plain HTML, CSS, JavaScript
- [Jikan API](https://api.jikan.moe/v4) for anime data
- Google Fonts (Outfit)

## Running Locally

Simply open `home.html` in your browser — no build step or server needed.
