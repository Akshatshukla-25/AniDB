# TradexW
My vision is to build an intelligent and user-friendly investment assistant that helps people make smarter and more informed financial decisions. By combining data analysis with AI, the platform aims to simplify complex stock market insights into clear and actionable recommendations like buy, hold, or avoid. I want to create a tool that is not only powerful but also accessible to beginners, reducing the guesswork and emotional bias involved in investing. Ultimately, the goal is to empower users to understand market trends better and invest with greater confidence through technology-driven insights.

# AniRate

AniRate is a React application built with Vite context.

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build the project for production:
   ```bash
   npm run build
   ```

4. Preview the production build locally:
   ```bash
   npm run preview
   ```

## Tech Stack
- React (Vite)
- Plain CSS
- React Router DOM
